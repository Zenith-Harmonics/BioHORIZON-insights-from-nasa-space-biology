What is BioHorizon?

BioHorizon is a simple web app that pulls space-biology data from open NASA sources, preprocesses it, and uses lightweight AI techniques to produce short, readable summaries. It’s designed for scientists, students and curious minds who want a fast entry point into complex research.

  • Simple UI: Minimal, responsive Bootstrap interface focused on exploration.
  • Data-first: Live ingestion from NASA open APIs (or exported datasets).
  • AI summaries: Concise, human-readable abstracts generated to speed understanding.
Built for Space Apps

This project was developed as an entry to the NASA Space Apps Challenge. The goal: prototype a useful tool in a short time that demonstrates impact and clarity.

Team & credits

BioHorizon was built by Zenith Harmonics for the NASA Space Apps Challenge. We focused on a minimal interface and proof-of-concept functionality that showcases how open data + lightweight AI can improve scientific discoverability.

Data used in this project originates from the NASA Open Science Data Repository (OSDR) , including datasets from CGene, ALSDA, and ESA. Please credit NASA OSDR and respect all data usage and attribution policies.
